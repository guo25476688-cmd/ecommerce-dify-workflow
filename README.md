# ecommerce-dify-workflow
# Multi-Platform Cross-Border E-Commerce Copywriting & Product Image Generation

An automated Dify workflow that takes product information as input and automatically generates marketing copy for Amazon, Shopee, TikTok, and SHEIN across multiple platforms and languages, while using AI to generate product images.

## Features
- Supports 4 major platforms: Amazon / Shopee / TikTok / SHEIN
- Automatic multi-language translation (Indonesian, Thai, Spanish, and more)
- Integrates Tavily real-time search for SEO trends
- Auto-generates product images (Pollinations.AI)

## How to Use
1. Import the `.yml` file into Dify
2. Install required plugins: OpenRouter, Tavily
3. Enter your product information and run
